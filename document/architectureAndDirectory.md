### Project Architecture &  Directory Overview
The project is structured into three distinct respositories/folders.

```text
/sccsms
├── /sccsmsmobile    # Mobile Client (iOS & Android)
├── /sccsmsserver    # Backend 
└── /sccsmsweb       # Web Client
```
| Module | Description | Tech Stack |
| :--- | :---: | ---: |
| sccsmsmobile | IOS&Android Client | React Native, TypeScript, React native paper |
| sccsmsserver | A monolithic backend server | Golang |
| sccsmsweb | Web Client | React, JavaScript, MUI |
